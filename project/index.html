  • This random music generator software generates music notes based on user input (music scale, tempo, etc) and common music theory. It is done in real time.   • Human history project called Digital Prosopography aims to turn human history digital text records to a relational web representation. It extracts the semantic meaning from text input, stores in rdf format with specific model, and present it as network graph of historical events.
