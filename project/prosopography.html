Digital Prosopography

Human history project called Digital Prosopography aims to turn human history digital text records to a relational web representation. It extracts the semantic meaning from text input, stores in rdf format with specific model, and present it as network graph of historical events.

After OCR of textbooks, digital data is parsed using GATE, an open software for text processing, and feed to openRDF for quad modeling. The research is done under Prof. Ichiro Fujinaga in DDMAL

Details of Event model:
Quad – Subject, Predict, Object, Context
Model 1: (EventID as Subject)
Subject – EventID
Predict – [hasSubject, hasAction, hasObject, hasDate, hasLocation]
Object – any object/other EventID
Context – Source (usually the name of the book)

Built in Node.js. http://github.com/LucilleH/lucilleh.com
